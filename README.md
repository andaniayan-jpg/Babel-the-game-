# Babel-the-game-
Babel is a project for #horizons, it is a game just like the arcade games of 90s, it has a starship whose aim is to combat with the alient ships in front of it. It has impressive graphics and sound effects.

**The development with 'index.html' was easier as i was familier with it. Since, i was not familier with JawaScript and css, I used AI for F  unctions, domain names and details, But the logic, math, and colours are my own work.**

# **You can get the files in repository further**



**I made this game as i wanted to learn basics of game development, I was familier with vibe coding websites and softwares, but not with gae development. Hence, i starte with small project.**

**This game contains a starship, and a loop of aliens(enemies) that are needed to be killed. The starship has 5 Lives(HPs) and as it kills the enemies, the score increases and so the level.**
